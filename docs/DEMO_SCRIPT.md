# Demo Script

1. Run `setup_windows.bat` or `python -m pip install -r requirements.txt`.
2. Start the backend with `run_windows.bat` or `python app.py`.
3. Open `http://127.0.0.1:5000`.
4. Log in with demo credentials in development mode if enabled.
5. Open **Import & Extraction**.
6. Load the production sample or upload `data/samples/professional_demo_network.cfg` after generating it with:
   `python -m wiguard.internal_tools.sample_data_generator`
7. Visit **Object Explorer** to show normalized evidence.
8. Visit **Threat Map** to show path/blast-radius readiness.
9. Visit **System Health** to show parser/dependency/storage readiness.
10. Visit **Report Builder** and export Professional JSON/HTML.
