# Reporting

WiGuard supports the existing JSON/HTML/PDF report templates and the new v5.16 professional normalized report exports.

## Professional exports

Routes:

- `/download/professional-analysis.json`
- `/download/professional-analysis.html`
- `/api/normalized-data`

These exports include:

- Summary
- Normalized assets
- Topology summary
- Security findings
- Risk score
- Redacted evidence preview
- Recommended actions
- Export metadata

Secrets are redacted by default.
